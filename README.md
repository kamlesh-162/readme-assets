## 🛠️ Skill Icons

Technology icons used in my GitHub README are provided by **Skill Icons**.

🌐 Website: https://skillicons.dev/

You can easily generate icons for your preferred programming languages, frameworks, tools, and technologies by selecting the skills you want.

### Example

```html
<img src="https://skillicons.dev/icons?i=cpp,python,java,mysql,javascript,git,vscode" />
```

## 🌊 Animated Banner (Capsule Render)

Create beautiful animated banners for your GitHub profile using **Capsule Render**.

🌐 Website: https://capsule-render.vercel.app/

Capsule Render allows you to customize:

* 🎨 Colors & gradients
* 🌊 Banner styles
* ✨ Animations
* 📍 Text alignment


### Example

```html
<img src="https://capsule-render.vercel.app/api?type=waving&height=220&color=0:6C63FF,25:00C9FF,50:00E5FF,75:00FFB3,100:92FE9D&text=Hi,%20I'm%20Kamlesh%20Shinde&fontColor=FFFFFF&fontSize=55&fontAlignY=35&animation=twinkling"/>
```
